## Portfolio site of myself

## [LIVE DEMO](https://naimburu.com/)

install [node](https://nodejs.org/en/)
install [git](https://git-scm.com/)

```shell
git clone https://github.com/nb256/naim-buru-com.git
cd naim-buru-com
npm install
npm run dev
```
